СЛАДКИЙ ЗАБЕГ — ОБНОВЛЕНИЕ 0.2.3 BETA
Исправление HTTP 404 в админ-рейтинге
=====================================

ЧТО ИСПРАВЛЕНО
---------------
1. Маршруты админ-панели перенесены непосредственно в src/worker.js.
2. wrangler.jsonc теперь запускает src/worker.js без дополнительных entrypoint-файлов.
3. Исправлен маршрут POST /api/admin/leaderboard/set.
4. Добавлен маршрут POST /api/admin/profile/sync.
5. Добавлен маршрут GET /api/admin/health для проверки сборки.
6. Глобальные цены магазина также перенесены в основной Worker.
7. В существующую админ-панель добавлено точное изменение рейтинга.

КАК УСТАНОВИТЬ
--------------
1. Сделайте резервную копию текущей папки zefirok-run.
2. Распакуйте содержимое этой сборки поверх папки проекта.
3. Убедитесь, что заменились файлы:
   - index.html
   - src/worker.js
   - wrangler.jsonc
4. Убедитесь, что появился файл:
   - migrations/0005_admin_profile_state.sql
5. Старые файлы src/entry.js и assets/global-shop-prices.js больше не нужны.
   В готовой сборке они уже удалены.

КОМАНДЫ В ТЕРМИНАЛЕ
-------------------
Перейдите в папку проекта:

cd /путь/к/zefirok-run

Проверьте изменения:

git status

Добавьте и отправьте обновление в GitHub:

git add -A
git commit -m "Fix admin rating API 0.2.3 beta"
git push

Примените новую D1-миграцию:

npx wrangler d1 migrations apply zefirok-rewards --remote

Опубликуйте Worker:

npx wrangler deploy

ПРОВЕРКА ПОСЛЕ DEPLOY
---------------------
Откройте в браузере:

https://zefirok-run.patokad6.workers.dev/api/admin/health

Ожидается JSON с:

"ok": true
"service": "zefirok-admin"
"version": "0.2.3 Beta"

Затем полностью закройте Telegram Mini App и откройте заново.
В админ-панели задайте рейтинг. Вместо HTTP 404 должно появиться:

"Серверный рейтинг установлен: ... очков."

ВАЖНО
-----
- SHOP_ADMIN_TELEGRAM_IDS должен содержать ваш числовой Telegram ID.
- TELEGRAM_BOT_TOKEN должен оставаться настроенным в Cloudflare.
- Миграция 0005 безопасна: используется CREATE TABLE IF NOT EXISTS.
- Уменьшение локальных ресурсов не понижает серверный минимум.
- Рейтинг задаётся точным числом и обновляет обе вкладки.

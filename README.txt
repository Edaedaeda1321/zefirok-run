СЛАДКИЙ ЗАБЕГ — 0.9.9 Beta

Основной файл:
  index.html

Файл является самодостаточным: изображения и игровая логика встроены внутрь HTML.
Папка assets/profile-buttons содержит отдельные исходники кнопки «Профиль».

НАСТРОЙКИ ОБНОВЛЕНИЯ

Откройте index.html в редакторе и найдите блок:
  НАСТРОЙКИ ВЕРСИИ, ОБНОВЛЕНИЙ И ГЛОБАЛЬНОГО СБРОСА ЭКОНОМИКИ

Основные параметры:
  GAME_VERSION — версия, отображаемая в профиле.
  ECONOMY_RESET_KEY — уникальный ключ глобального сброса экономики.
  UPDATE_NOTICE_KEY — уникальный ключ окна обновления.
  UPDATE_NOTICE_ENABLED — включение или отключение окна обновления.
  UPDATE_NOTICE_TITLE — заголовок окна.
  UPDATE_NOTICE_TEXT — текст окна.
  UPDATE_NOTICE_BUTTON — текст кнопки.

СБРОС ЭКОНОМИКИ

Чтобы при следующем запуске сбросить прогресс всех игроков, измените
ECONOMY_RESET_KEY на новую уникальную строку. Например:
  economy-reset-2026-08-01

Сброс произойдёт у каждого игрока один раз при первом запуске новой версии.

ОБНОВЛЕНИЕ БЕЗ СБРОСА

Не меняйте ECONOMY_RESET_KEY. Измените GAME_VERSION, UPDATE_NOTICE_KEY
и текст уведомления.
